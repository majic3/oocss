This is better than using a fully js reliant tab system (it works sans js albeit with updating head navigation of tabs) often JS is used to achieve results which css can do. The mini piece of script below uses jquery to update the remove the class current from but OOCSS is JS agnostic. The methodology of OOCSS does not seek to favour the use of any one js framework over any other - or in other words you can use the framework of your choice.
