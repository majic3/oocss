Tab Modules

two lists hd (head list navigation) & bd (body content)

a sweet little setup which achieve an effect commonly used on the web (bbc & yahoo) both have such a thing on respetive home pages. The class current needs to be remove and reapplied to new current using some js. This is better than using a fully js reliant tab system (it works sans js albeit with updating head navigation of tabs) often JS is used to achieve results which css can do. The mini piece of script below uses jquery to update the remove the class current from but OOCSS is JS agnostic. The methodology of OOCSS does not seek to favour the use of any one js framework over any other - or in other words you can use the framework of your choice. see the tabs branch
